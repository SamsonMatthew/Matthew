HIDIT v1.0.0

Basically this App hides your secret message inside the Image you choose.Before downloading make sure you read all the information in Helper section.

This App only works for Android for now.
Click the Button below to download it

Helper Section:

Installing:
Open the downloaded file from Downloads->It may Prompt you to "allow to install unknown apps from this source": Allow it->Install->OPEN->"Allow HidIt to acess your photos,media and files on your device?": Click Allow ->Now the App is successfully Installed.

How To Create It?
Click on create button or open the navigation drawer and select create.Click the Button to select the image on which you want to hide a message.
Remember to select the image with JP[E]G or PNG format only. After selecting image, Your image name will be appeared below the Button.Next write the secret message, add the password to it and Give a name for the image you created.Click the Button to Create.Your image will be saved in the folder named HIDIT.

Warning:While sharing the image in any media remember to share with the "Original Resolution".

How to Reveal it?
Click on Reveal button or open the navigation drawer and select Reveal.Click the Button to select the image on which secret message is hidden.
Remember to select the Image which has secret message.After selecting the image make sure you enter the correct password and Click the Reveal button to reveal the secret message

Why to give permission?
When you choose the Image to hide a secret message or reveal it, This App needs a special permission to access Device Storage.
